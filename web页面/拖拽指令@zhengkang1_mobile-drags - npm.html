    Vue 拖拽

    下载

    npm install @zhengkang1/mobile-drags

    引入

    在 main.js 中引入

    import drags from "@zhengkang1/mobile-drags/lib"

    使用

    //在main.js中注册全局自定义指令
Vue.directive('drag', drags)

//在页面中
   <div class="out" v-drag></div>
