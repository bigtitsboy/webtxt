Array数组的splice()方法，也是一个非常强大的方法，它的作用是:删除、插入、替换

需要注意的是： splice()方法是直接修改原数组的


一、删除的用法

语法: array.splice(starti,n);


starti 指的是从哪个位置开始(不包含starti)

n指的是需要删除的个数

  1. <script>
  2.     var array=[1,2,3,4,5];
  3.     array.splice(3,2);
  4.     console.log(array);
  5. </script>


结果： [1,2,3]


这里有个小拓展：其实被删除的元素可以用一个变量接收的，这个接收的变量可以作为拼接数组来使用

  1. <script>
  2.     var array=[1,2,3,4,5];
  3.     var deletes =array.splice(3,2);
  4.     console.log(deletes);
  5.     console.log(array);
  6. </script>
结果： [4,5]   [1,2,3]
 

我们将删除后的元素在拼接回原来的数组
 

  1. <script>
  2.     var array=[1,2,3,4,5];
  3.     var deletes =array.splice(3,2);
  4.     console.log(deletes);
  5.     console.log(array);
  6.     array=array.concat(deletes);
  7.     console.log(array);
  8. </script>
结果：  [4,5]   [1,2,3]  [1,2,3,4,5]
 

二、插入的用法
 

 

语法：array.splice(starti,0,值1，值2...);
 

 

starti: 在哪个位置插入，原来starti位置的值向后顺移
 

0：表示删除0个元素，因为插入和替换都是由删除功能拓展的。
 

值1，值2：需要插入的值
 

 

  1. <script>
  2.     var array=[1,2,3,4,5];
  3.     array.splice(2,0,123,456);
  4.     console.log(array);
  5. </script>

结果： [1,2,123,456,3,4,5]
 

 

 

 

三、替换的用法
 

语法:array.splice(starti,n,值1，值2);
 

原理和插入的用法相同
 

实际是就是：在starti的位置删除n个元素，然后在这个位置插入值1，值2,就可以起到替换
 

原来被删除的值
 

 



  1. <script>
  2.     var array=[1,2,3,4,5];
  3.     array.splice(2,2,123,456);
  4.     console.log(array);
  5. </script>


 


 
 

 结果：[1,2,123,456,5] 


 
 
 


 
 

 

总结:
 

splice()方法实际是一个删除数组元素方法，但可以拓展出插入，和替换两个用法
